# Recommendation and Statistics Prototype

This folder contains a lightweight prototype for a future recommendation and analytics layer.

## What it contains
- A simple rule-based recommendation engine
- A lightweight statistics summarizer
- A JSON API contract you can later connect to the Laravel app

## How to run
From this folder:

```bash
python app.py
```

Then open:

```text
http://127.0.0.1:5000/recommendations
http://127.0.0.1:5000/statistics
```
